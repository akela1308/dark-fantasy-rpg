# ПЛАН НА СЕГОДНЯ — Dark Fantasy RPG

## ЧТО УЖЕ СДЕЛАНО (Claude создал прямо сейчас)
Весь фундамент проекта лежит в папке `dark-fantasy-rpg/`.
Осталось запустить и опубликовать.

---

## ШАГ 1 — Установить Node.js (если нет) [5 мин]

Открой терминал и проверь:
```
node --version
```
Если выдаёт версию 18+ — переходи к шагу 2.
Если нет — скачай с https://nodejs.org (кнопка LTS).

---

## ШАГ 2 — Установить зависимости и запустить игру [5 мин]

В терминале перейди в папку проекта:
```
cd ~/Desktop/Dark\ Fantasy/dark-fantasy-rpg
npm install
npm run dev
```
Открой браузер: http://localhost:5173
Должна появиться заставка с кнопкой "НАЧАТЬ" и рабочий бой.

---

## ШАГ 3 — Создать репозиторий на GitHub [10 мин]

1. Зайди на github.com → войди в аккаунт
2. Нажми "+" → "New repository"
3. Имя: `dark-fantasy-rpg`
4. Visibility: **Public** (обязательно для GitHub Pages)
5. НЕ ставь галочки (Initialize, .gitignore, license)
6. Нажми "Create repository"
7. Скопируй ссылку репозитория (формат: https://github.com/ВАШ_НИК/dark-fantasy-rpg.git)

---

## ШАГ 4 — Подключить проект к GitHub и задеплоить [10 мин]

В терминале (находясь в папке dark-fantasy-rpg):
```
git init
git add .
git commit -m "Initial commit: project foundation"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/dark-fantasy-rpg.git
git push -u origin main
```
Замени ВАШ_НИК на свой GitHub username.

---

## ШАГ 5 — Включить GitHub Pages [5 мин]

1. Открой репозиторий на github.com
2. Settings → Pages
3. Source: выбери ветку `gh-pages` (появится через ~2 мин после push)
4. Save

Через 2–3 минуты игра будет доступна по ссылке:
https://ВАШ_НИК.github.io/dark-fantasy-rpg/

---

## ШАГ 6 — Проверить что бой работает [15 мин]

Открой локально (localhost:5173) и пройди полный цикл:
- [ ] Заставка открывается
- [ ] Нажимаю "НАЧАТЬ" → переход в бой
- [ ] Вижу своих юнитов (синие) и врагов (красные)
- [ ] Кликаю по врагу — он получает урон
- [ ] Ход переходит к врагу автоматически
- [ ] Враги ходят сами
- [ ] Когда все враги мертвы — экран победы
- [ ] Если герой мёртв — экран поражения
- [ ] Кнопка "Попробовать снова" — перезапускает бой

После выполнения всех шагов у тебя будет:
✓ Рабочий браузерный бой с 4 юнитами vs 4 врагами

✓ Боевой лог
✓ Победа / поражение
